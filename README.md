# Knuckle Joint Assembly

A CAD design and structural analysis project of a **Knuckle Joint** developed using **Autodesk Inventor** and **ANSYS Workbench Student**.

---

## 📖 Project Overview

The Knuckle Joint is a mechanical joint used to connect two rods subjected to tensile loads while allowing angular movement. This project involves creating the complete CAD model, assembling all components, and performing static structural analysis to evaluate the joint's performance under applied loading conditions.

---

## 🛠️ Software Used

- Autodesk Inventor
- ANSYS Workbench Student

---

## ⚙️ Components

- Eye End
- Fork End
- Knuckle Pin
- Collar
- Taper Pin
